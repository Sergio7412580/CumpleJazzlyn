# Instrucciones de Música de Fondo

Para colocar tu música de fondo favorita:
1. Copia tu archivo de música K-Pop en formato MP3 en esta carpeta.
2. Renombra el archivo a `bg-music.mp3`.
3. El reproductor web cargará automáticamente tu canción.

*Nota:* Si el archivo no existe o falla al cargarse, la página web tiene un sintetizador dinámico programado en Web Audio API que generará un tema Synthwave/K-Pop retro de forma automática al presionar "Play".
